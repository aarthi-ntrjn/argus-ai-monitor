Built for the intensity of AI development

In the AI era, protecting your attention is critical. When you're building with intensity, the complexity of modern AI-powered development adds up fast.

Multiple sessions + Multiple tools + Multiple repositories = ALT-TAB hell

Argus was built to make working across sessions seamless, so you can stay in flow and focus on what you're building.

How It Works

Three steps to full visibility into your AI coding sessions.

1

Install

Run npx argus-ai-hub in any terminal. No global install needed; it starts up instantly.

2

Add Repositories

Navigate to http://localhost:7411. Add a repository and Argus immediately starts monitoring all running Claude Code and Copilot CLI sessions within it.

3

Launch with Argus

Stream output live, send prompts, kill runaway sessions, or focus your terminal. Stay in control without leaving the dashboard.

Everything you need to stay in control

Monitor

See all your running AI sessions at a glance. Each session card shows the tool (Claude Code or Copilot CLI), current status, active model, process ID, and elapsed time. No more switching terminal windows to check what is running.

Session Output

Stream session output in real time with color-coded message types: YOUR prompts, AI responses, TOOL calls, and RESULTS. Switch between focused mode (AI responses only) and verbose mode (full trace) without leaving the browser.

Control

Send follow-up prompts directly to any running session, kill runaway agents before they rack up tokens, and focus your terminal on the session you care about most. All from a single keyboard-friendly interface.
