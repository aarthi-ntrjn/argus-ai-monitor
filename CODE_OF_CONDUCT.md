# Code of Conduct

## Sole Maintainer Project

Argus is a personal project maintained by a single developer. This repository is **not accepting external contributions** at this time.

## What This Means

- Pull requests from external contributors will not be accepted
- You are welcome to fork the project for your own use under the terms of the [MIT License](LICENSE)
- Bug reports via Issues are appreciated but may not be addressed on any particular timeline

## Respectful Communication

If you open an issue or otherwise interact with this project, please be respectful and constructive. Harassment, abusive language, or spam will result in being blocked.

## Contact

For questions, open an issue on the [GitHub repository](https://github.com/aarthi-ntrjn/argus/issues).
