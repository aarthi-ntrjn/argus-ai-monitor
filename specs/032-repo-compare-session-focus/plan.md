# Implementation Plan: Repository Compare Link and Session Focus Button

**Branch**: `032-repo-compare-session-focus` | **Date**: 2026-04-13 | **Spec**: `specs/032-repo-compare-session-focus/spec.md`

## Summary

Two UI productivity features for the Argus dashboard. Feature 1 adds a GitHub compare link icon to the repository card header, constructed from the repo's `origin` remote URL and current branch. Feature 2 adds a Focus button to the session card that triggers a backend `POST /focus` endpoint to bring the terminal window hosting the AI process to the foreground using OS-native APIs (PowerShell, AppleScript, wmctrl).

Both features touch the same full stack: DB schema (new `remote_url` column), backend service layer, REST API routes, frontend types, and React components.

## Technical Context

**Language/Version**: TypeScript (Node.js 20 backend, React 18 frontend)

**Performance Goals**: API responses within 500ms p95 (§VIII)

**Scale/Scope**: ≥10 concurrent sessions monitored simultaneously
