# Implementation Plan: Bulk Repository Import (Scan Folder)

**Branch**: `003-remove-repository` | **Date**: 2026-04-01 | **Spec**: [spec.md](./spec.md)
**Input**: Feature specification from `/specs/003-remove-repository/spec.md`

## Summary

Add a "Add Multiple Repositories" action to the Argus dashboard that opens the native OS folder picker, recursively scans the selected folder for git repositories, and bulk-adds all newly found repos (skipping already-registered ones). One user story, two new endpoints, minimal frontend changes.

## Technical Context

**Language/Version**: TypeScript (Node 20 backend, React 18 frontend)
**Primary Dependencies**: Fastify (backend API), React Query (frontend), fs/path (Node built-ins for scan)
**Storage**: SQLite via better-sqlite3 — `repositories` table already exists
**Testing**: Vitest (backend), Playwright (E2E)
**Target Platform**: Local developer machine (Windows/macOS/Linux)
**Project Type**: Web application (backend + frontend)
**Performance Goals**: Scan and add 50 repos in < 5 seconds
**Constraints**: No external deps for scanning — use Node.js `fs.readdirSync` recursively
**Scale/Scope**: Single local developer, ≥10 concurrent sessions

## Constitution Check
