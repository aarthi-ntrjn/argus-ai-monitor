# Implementation Plan: GitHub Feedback Links

**Branch**: `046-github-feedback` | **Date**: 2026-04-19 | **Spec**: [spec.md](./spec.md)
**Input**: Feature specification from `/specs/046-github-feedback/spec.md`

## Summary

Add a "Feedback" dropdown button to the navigation header on every page of the Argus dashboard. The dropdown contains two items: "Report a Bug" and "Request a Feature". Each item opens a pre-filled GitHub new issue form on the Argus public repo in a new browser tab. No backend, no authentication, and no new infrastructure is required.
