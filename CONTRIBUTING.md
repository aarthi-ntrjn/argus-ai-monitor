# Contributing

Thank you for your interest in Argus!

## Current Status

Argus is currently in early development. **We are not accepting external contributions
at this time**, but we welcome bug reports and feature requests via
[GitHub Issues](https://github.com/aarthi-ntrjn/argus/issues).

## Reporting Bugs

If you find a bug, please open an issue with:

- A clear description of the problem
- Steps to reproduce
- Expected vs actual behavior
- Your OS and Node.js version

## Requesting Features

Feature requests are welcome as GitHub Issues. Please describe the use case and why the
feature would be valuable.

## Security Issues

For security vulnerabilities, **do not open a public issue**. Instead, use
[GitHub's Private Vulnerability Reporting](https://github.com/aarthi-ntrjn/argus/security/advisories/new).
See [SECURITY.md](SECURITY.md) for details.

## For Maintainers

Internal contributor documentation (architecture, dev setup, testing, API reference) is
in [docs/README-CONTRIBUTORS.md](docs/README-CONTRIBUTORS.md).
